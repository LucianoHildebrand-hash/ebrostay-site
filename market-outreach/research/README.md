# Research

Drop the market-research spreadsheet here (e.g. `market-research.xlsx`).

This holds the inputs behind the outreach effort: company segments, candidate
industries, employer size, relocation patterns into Zaragoza, and any scoring
used to prioritise targets.

> Note: GitHub does not render `.xlsx` previews. If you want the data to be
> diff-able and viewable directly on GitHub, also export a CSV (one per sheet)
> alongside the workbook.
